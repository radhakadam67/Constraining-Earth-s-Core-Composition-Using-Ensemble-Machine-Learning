Algorithm Design

Why Three Different Algorithms?

Different elements follow different physical laws, so each gets the algorithm that best matches its behaviour — a novel approach no prior study used.

🌲
Random Forest
Fe, Ni — Major Metals
Ensemble of 200–300 decision trees. Captures non-linear metal-metal interactions through conditional branching — similar to how different metals in an alloy affect each other's properties.
📐
Ridge Regression
O, S, Si — Light Elements
L2-regularised linear model. These elements follow Henry's Law thermodynamics, creating clean linear relationships in log-space that Ridge captures efficiently without overfitting.
🔁
Gradient Boosting
Cr, Co, W, Mo, V, Nb — Traces
Sequential error correction over 200 iterations. Trace elements have high signal-to-noise ratios; GB iteratively filters out measurement noise to find genuine geochemical patterns.
Novel Contributions

What's New in This Work

01
First Complete 14-Element Core Prediction
All prior studies predicted only 60–80% of core composition. This framework covers 100% with perfect mass balance closure (1.000 ± 0.001).
02
Element-Specific Algorithm Optimisation
Novel methodology assigning different ML algorithms based on each element's thermodynamic behaviour — replacing the one-size-fits-all approach of prior work.
03
Fugacity-Aware Feature Engineering
Oxygen, sulphur, and carbon fugacity controls integrated as ML features for the first time, enabling direct application to modern experimental studies.
04
Thermodynamic Constraint Enforcement
Post-prediction normalisation enforces mass balance, charge balance, and Wagner element-interaction constraints — ensuring all predictions are geochemically valid.
05
First Positive ML Predictions for Trace Elements
Tungsten (R²=0.345) and Niobium received their first-ever successful ML predictions — elements previously considered too noisy to model computationally.
Data

The Experimental Dataset

High-pressure lab experiments recreating conditions 2,900+ km underground. Each data point required crushing samples to extreme pressures using diamond anvil cells or multi-anvil presses.

Experimental Samples 196
Variables (after cleaning) 137
Pressure Range 1–100 GPa
Temperature Range 1773–5700 K
Data Completeness 98.7%
Concentration Challenge

Element concentrations span 6 orders of magnitude — the biggest ML challenge in this dataset.

Iron (Fe) 70–95%
Nickel (Ni) ~5%
Silicon (Si) ~1–3%
Niobium (Nb) <0.01%

This 6-order-of-magnitude spread is why traditional stats fail — and why element-specific ML is needed.
